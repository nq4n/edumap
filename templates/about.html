About This Project

Purpose

This project is an interactive web application designed as a "University Building Lab Locator". It presents users with a visual SVG map of a building floor plan. Users can interact with the map by clicking on specific rooms or labs, or by using a search bar to find locations based on room number, description, or type. Selecting a room displays detailed information about it.

Key Features

  • Interactive SVG Map: Displays a clickable vector map (lab.svg) of the building layout.
  • Room Detail Display: Shows information (name, location, description, capacity, equipment) fetched from a data source (data.xlsx) when a room is clicked on the map or selected from search results.
  • Search Functionality: Allows users to search for rooms by number, description keywords, or room type, with live results appearing below the search bar.
  • Visual Highlighting: Clearly highlights the selected room on the SVG map using styling changes (e.g., green fill).
  • Data-Driven: Room details are managed externally in an Excel file (data.xlsx) and loaded by the backend.
  • Responsive Feedback: Provides loading indicators and error messages to the user during interactions.
